### Overview
BGG Hover is a Chrome extension I developed to speed up browsing on BoardGameGeek, GeekLists, and other sites with BGG links. Instead of opening a new tab for each game, the extension shows a quick hover preview with core details like designer, rating, and weight.

This free extension is downloaded by almost 200 users on the Chrome Web Store!

*This extension is unfortunately no longer working due to a change to the BoardGameGeek API that requires individual users to request API keys. I plan on rebuilding this application at some point, but it would require a full reworking of the app.*
